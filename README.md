# DB_project_react


## Usage
 - clone project
 - cd DB_React
 ### back
 - cd back 
 - run ```
       yarn
       ``` 
in terminal.
 - run ```
       yarn start
       ``` 

### front
 - cd front 
 - run ```
       yarn
       ``` 
in terminal.
 - run ```
       yarn start
       ```
